# Apex API — LLM Token Usage Tracker

Track your API spending across Groq, Cohere, OpenAI, Anthropic, and Ollama in real-time.

## Overview

Apex API is a SaaS platform that monitors your LLM API usage and costs automatically. Install the browser extension, and it tracks every token you use across multiple APIs, providing detailed analytics and cost breakdowns on a unified dashboard.

## Features

- **Real-time tracking** — Passively monitors all API calls
- **Multi-API support** — Groq, Cohere, OpenAI, Anthropic, Ollama
- **Cost analytics** — Token usage, spend breakdown by API, daily trends
- **Browser extension** — Lightweight, non-intrusive tracking
- **Unified dashboard** — View all APIs in one place

## Tech Stack

**Backend:**
- FastAPI (Python)
- PostgreSQL
- JWT authentication
- Deployed on Render
